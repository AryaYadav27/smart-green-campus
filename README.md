## Problem
Students study sustainability theoretically, but lack motivation and academic relevance to practice it in daily life.

## Solution
Smart Green Campus is a web-based application that converts daily sustainable behaviors into an EcoScore to motivate students through feedback, recognition, and optional academic incentives.

## Key Features
- EcoScore-based sustainability measurement
- Carbon footprint awareness
- Sustainable behavior suggestions
- Student-focused design

## Tech Stack
- Python
- Streamlit
- Google Cloud (planned)
